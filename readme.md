##主题：方寸屏间，万里山河
###网站主题：中国印象

##设计理念 1.使用户多层次多交互地欣赏丰富的中国风景。

2.给予用户非常丝滑的浏览体验。

##设计特点 1.总体使用简约清晰的页面设计。

2.使用标准中国地图进行用户交互

3.网页交替使用了竖式滚动和横向滚动，提升用户的体验感。

4.通过特殊的滚动效果，增强图片展示的呈现效果。

5.通过图片水墨风格转化提升用户体验。

##代码实现
使用 Gsap 以及其免费开源子库实现滚动动画。

使用 lenis 的支持库来做到平滑滚动，优化滚动的体验感。

使用 SVG 格式的标准中国地图作为子页面索引。

本网页没有使用框架库。

##网页呈现
###主页面
封面展示网页主题以及可交互中国地图。

向下滚动后先呈现一个视频，之后不断滚动文字出现并且在固定视角中心，背景使用卡片风格的图片呈现一种流动的效果。

再往下滑，呈现一个图片分裂为三个卡片的效果，卡片分别展示人文，自然，城市三个主题。

本来每张卡片是一种主题图片展览的索引（由于时间关系，仍未完成），

再往后滚动变为横向滚动，分不同意象进行精选图片的展示以及对中国地大物博的介绍。（由于时间关系，只做了 2 个部分来演示，并且尚未调整好横向滚动的导航栏功能）

最后是总结以及页脚。

###分页面
本队完成了除了香港澳门以外其他 32 个省级区域（自治区，直辖市）的分网页设计。

首页呈现地区名字以及地方地图，整体风格与主页面交互地图一致。

向下滚动呈现地区介绍，然后一张图片从视角中间从无不断放大直到占满屏幕。

这张图片中呈现古诗（现代诗），并且可以通过按钮实现水墨风转变，丰富用户交互体验。

随后是横向滚动展示图片，鼠标悬浮可以显示地点以及对风景的简要描述。

横向展示结束后，竖向展示图片，通过 js 以及 gsap 库实现幻灯片播放效果以及左侧进度栏效果。

最后是页脚。

（由于时间原因，部分分页面的文字以及水墨风交互图片并不完整）
